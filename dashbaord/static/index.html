ビギクラ!

ビギクラ! は初心者向けの脆弱性学習アプリです。ビギクラ! は脆弱性のある掲示板アプリと、実行される SQL 文や実行結果を表示するモニターのセットです。実行される SQL 文を見ながら攻撃をしてみることで、誰でも簡単に攻撃を体験することができます。

手順1 モニターを起動

モニターは、アプリで実行された SQL 文とその実行結果が表示されます。まずはモニターを起動しましょう!

モニターを起動 >>

手順2 掲示板を起動

何の特徴もない会員制の掲示板です。ログイン ID に admin パスワードに admin でログインができます。

まずはモニターを見ながら、ログインした時、記事を投稿した時、表示した時など、どのような SQL 文が実行されるか? その結果がどうなるのかを確認しましょう!

掲示板を起動 >>

手順3 攻撃しよう!

掲示板の機能を一通り試したら、今度は攻撃にチャレンジしてみましょう! 少し工夫をするだけで、admin のパスワードを知らなくてもログインできるかもしれません。他にもいろいな脆弱性がありますのでぜひ試してみてください!
