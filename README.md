# ビギクラ!
## 実行方法
### サービスを利用する
ビギクラ! は Arukas (beta) を利用したサービスを提供しています。

https://begicra.arukascloud.io/

にアクセスしてください。


### Docker Image を利用する
Docker Image を用意しています。masakura/begicra から最新のタグを取得して実行してください。

https://hub.docker.com/r/masakura/begicra/
